Featured Case Study

Scalable, Adaptive Anomaly Detection for Housing Streams

R | Shiny | isotree | torch (optional) | renv | testthat

A drift-aware anomaly detection system for high-dimensional housing/real-estate streams. Simulates evolving market regimes, expands features (100 → 10,000), applies rolling PCA, and detects anomalies with a robust adaptive ensemble and interpretable back-mapped explanations.

Highlights
  • Adaptive ensemble (MAD baseline + Isolation Forest + optional Autoencoder)
  • Concept drift injection + detection
  • High-dimensional scaling with structured stress tests
  • Explainability via PCA back-mapping
  • Shiny live demo + CI + tests
